考虑实际情况，暂时做如下参数设定（其中扫频宽度还需微调）：
1.扫频宽度：0-2e10
2.脉冲宽度：1e-8=0.01us
3.脉冲周期：1e-6=0.1us
4.采样频率：5e11 （5倍载频频率）
因此每个脉冲周期内有5e4个采样点，对地扫描9×9方格，共
81帧的数据，最后结果为81×50000的矩阵

由于脉宽为τ=0.01us，所以径向距离最近可达到τ*c/2=1.5米，
设车高度为1.4米，所以地面距离为sqrt(1.5^2-1.4^2)= 0.54m
应该符合实际情况

由于脉冲周期T = 0.1us，若不考虑距离模糊情况（在该周期发出
脉冲但在下一个脉冲周期内收到）则可测最远距离为15米，
最近距离为0.54米，足够汽车做出刹车等响应

目前设置的是在第20帧的时候扫描到了物体
